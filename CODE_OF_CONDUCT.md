# Code of Conduct

## Short Version

Be respectful. Be constructive. Don't waste other people's time.

## Longer Version

SeeseoCrawler is maintained by [SEObserver](https://www.seobserver.com) and open to contributions from everyone. We expect participants to:

- **Be respectful.** Disagreement is fine. Personal attacks, insults, and trolling are not. Critique code, not people.
- **Be constructive.** If you open an issue saying "this is broken", include steps to reproduce. If you review a PR, explain *why* something should change, not just that it should.
- **Stay on topic.** This project is about crawling websites and extracting SEO signals. Keep discussions relevant.
- **Accept feedback gracefully.** Maintainers may reject PRs or close issues. That's not personal — it's curation.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language
- Publishing others' private information
- Spam, self-promotion, or off-topic solicitation
- Deliberately derailing discussions

## Enforcement

Instances of unacceptable behavior can be reported to **contact@crawlobserver.com**. Maintainers will review and take appropriate action, which may include warning, temporary ban, or permanent ban from the project.

## Scope

This code of conduct applies to all project spaces: GitHub issues, pull requests, discussions, and any other channel associated with SeeseoCrawler.
